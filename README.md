# Coffee Store Client

A coffee e-commerce SPA built with React, featuring product browsing, cart management, and CRUD operations with a connected backend server.

## Features

- **Product listing**: Browse coffees with details and ratings
- **CRUD operations**: Add, update, and delete coffee products
- **Cart management**: Add/remove items with SweetAlert2 confirmations
- **Responsive design**: Tailwind CSS + DaisyUI with mobile-first approach

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + Vite 4 |
| Routing | React Router DOM 6 |
| Styling | Tailwind CSS 3 + DaisyUI 3 |
| UI | SweetAlert2 |

## Getting Started

```bash
npm install
npm run dev
```

## API

Connects to the [coffee-store-server](https://github.com/tarekul42/coffee-store-server) backend.
